## 1. 제품 개요

### 1.1 한 줄 정의
> 생년월일시를 입력하면 사주팔자를 자동 산출하고, 오행·천간·지지 분석부터 대운·세운·일운까지 종합 명리 해석을 제공하는 한국어 웹 기반 사주 프로그램.

### 1.2 제품 비전
전통 명리학의 깊이를 현대 사용자 경험으로 번역한다. 복잡한 사주 이론을 전문가 수준으로 정확하게 계산하되, 누구나 직관적으로 읽을 수 있는 해석 결과를 제공한다. 디자인은 Apple 브랜드 철학을 재해석한 미니멀 동양 미학을 따른다.

### 1.3 목표 사용자

| 페르소나 | 설명 | 주요 니즈 |
|---------|------|---------|
| **일반 탐색형** | 사주에 처음 관심을 가진 20–40대 | 쉬운 입력, 이해하기 쉬운 결과 |
| **관심 심화형** | 사주를 어느 정도 아는 30–50대 | 정확한 계산, 오행·용신 분석 |
| **전문가 검증형** | 명리학 공부 중인 사용자 | 원국 상세 정보, 신살·십신 등 세부 정보 |

### 1.4 핵심 가치 제안
- **정확성:** 만세력(萬歲曆) 기반 정밀 사주 산출. 절입 시각 반영.
- **완결성:** 원국 분석 → 오행·용신 → 대운·세운·일운까지 하나의 흐름.
- **가독성:** 한자와 한글을 디자인 원칙에 따라 병기하여 전문성과 접근성 동시 달성.
- **신뢰감:** 광고 없는 순수 정보 제공, 근거 있는 해석 텍스트.

## 3. 기능 상세 요구사항

### F1. 사주 입력 및 산출

#### F1-1. 생년월일시 입력
- 생년월일을 숫자 직접 입력 또는 달력 피커로 선택 가능해야 한다.
- 지원 연도 범위: 1900년 1월 1일 ~ 2100년 12월 31일.
- 태어난 시간은 12지지 시(時) 단위(자시·축시·…·해시) 선택과 정확한 시각 입력 두 가지 방식을 모두 지원한다.
- 시간 모름 선택지를 제공한다. 시간 미입력 시 시주(時柱) 미산출 상태로 결과를 표시하되, 일부 기능 제한 안내를 표시한다.

#### F1-2. 성별 입력
- 남성 / 여성 선택 (대운 역순 계산에 영향).
- 성별 미선택 시 대운 기준 안내 메시지 표시.

#### F1-3. 양력/음력 선택
- 양력(陽曆) 기본값.
- 음력 선택 시 윤달 여부 선택 옵션 추가 노출.
- 음력 입력값을 내부적으로 양력으로 변환하여 계산.

#### F1-4. 사주팔자 자동 산출
- 입력 완료 즉시(또는 "사주 보기" 버튼 클릭 시) 팔자 산출.
- 절입(節入) 시각을 반영한 정밀 만세력 계산. (예: 입춘 전 출생자는 전년도 천간 적용)
- 야자시(夜子時) / 조자시(早子時) 처리 옵션 설정 제공.
- 계산 결과: 연주·월주·일주·시주의 천간·지지 8글자.

### F2. 원국(原局) 분석

#### F2-1. 사주명반(命盤) 표시
- 4개 기둥(연·월·일·시)을 열 형태로 시각화.
- 각 기둥에 천간(위)·지지(아래) 한자를 크게 표시.
- 각 기둥 위에 연주·월주·일주·시주 한글 레이블 표시.
- 일주(日柱) 강조 처리 — 일간(日干)이 사용자 본인 기준.
- 천간·지지 각 글자에 오행 색상 시각 표시(작은 도트 또는 배경 틴트).
- 명반 하단에 오행 비율 바 표시.

#### F2-2. 천간(天干) 분석
- 각 천간 글자(甲乙丙丁戊己庚辛壬癸)의 오행·음양 정보 표시.
- 일간 기준 각 천간의 십신(十神) 관계 표시.
- 천간 합(天干合)·충(沖) 자동 감지 및 표시.

#### F2-3. 지지(地支) 분석
- 각 지지 글자(子丑寅卯辰巳午未申酉戌亥)의 오행·음양·계절 정보 표시.
- 지지 삼합(三合)·방합(方合)·육합(六合)·충(沖)·형(刑)·파(破)·해(害) 자동 감지 및 표시.
- 감지된 합·충·형 관계는 시각적으로 연결선 또는 배지로 표시.

#### F2-4. 지장간(支藏干) 표시
- 각 지지의 지장간(여기·중기·정기) 표시 (접기/펼치기 가능).
- 지장간 기준 십신 관계 추가 표시.

#### F2-5. 십신(十神) 분석
- 일간 기준 비견·겁재·식신·상관·편재·정재·편관·정관·편인·정인 10개 분류 자동 산출.
- 십신별 한글 해석 텍스트 제공 (성격·역할·의미).
- 특정 십신 과다·부재 시 강조 안내.

---

### F3. 오행(五行) 분석

#### F3-1. 오행 분포 시각화
- 원국 8글자(천간 4 + 지지 4) 기준 오행 분포 비율 계산.
- 木·火·土·金·水 5개 항목을 가로 막대 그래프로 시각화.
- 일간의 오행이 앱 전체 강조색(--color-primary)으로 표시.
- 수치는 퍼센트(%) 한글 병기.

#### F3-2. 강약 분석
- 일간(日干)의 신강(身强) / 신약(身弱) 판단 결과 표시.
- 월지(月支) 기준 득령(得令) 여부, 주변 천간·지지의 생조(生助) 계산.
- 강약 판단 근거를 한글 텍스트로 요약 제공.

#### F3-3. 용신(用神) 도출
- 억부법(抑扶法) 기준 용신·희신·기신·구신·한신 도출.
- 용신 오행과 대표 천간·지지 한자를 함께 표시.
- 용신 적용 방향 (생활·직업·환경 등) 한글 해석 제공.

---

### F4. 대운(大運) 분석

#### F4-1. 대운 수 계산
- 성별과 연간(年干)의 음양에 따른 순행·역행 자동 판별.
- 생일로부터 가장 가까운 절기까지의 날수로 대운 수(起大運數) 계산.
- 대운 수 계산 근거 날수 표시.

#### F4-2. 대운 타임라인
- 10년 단위 대운을 가로 타임라인으로 시각화.
- 총 10개 이상의 대운 표시 (약 100세 이상 커버).
- 현재 나이에 해당하는 대운 강조 표시.
- 각 대운의 천간·지지 한자와 시작 나이 표시.
- 대운 클릭 시 해당 대운 상세 해석 패널 열림.

#### F4-3. 현재 대운 해석
- 현재 진행 중인 대운의 천간·지지 분석.
- 원국과의 합·충·형·파·해 관계 분석.
- 대운 기간 주요 흐름 한글 해석 텍스트.

---

### F5. 세운·월운·일운 분석

#### F5-1. 현재 세운(歲運) 분석
- 현재 연도의 태세(太歲) 천간·지지 자동 반영.
- 원국 및 대운과의 상호 관계 분석.
- 년도 선택기로 과거·미래 세운 조회 가능 (현재 ±10년).

#### F5-2. 월운(月運) 분석
- 현재 월의 월건(月建) 분석.
- 세운·대운·원국과의 관계 해석.
- 월 선택기로 해당 연도 12개월 월운 순회 가능.

#### F5-3. 일운(日運) 달력
- 현재 월을 달력 형태로 표시.
- 각 날짜에 일진(日辰) 천간·지지 한자 소형 표시.
- 날짜 클릭 시 해당 일진과 원국의 관계 간략 해석.
- 길일(吉日) / 흉일(凶日) 표시 (용신 오행 기준).

### F6. 종합 해석 리포트

각 해석 항목은 원국·오행·십신 분석 결과를 종합한 한글 텍스트로 제공한다.  
해석 텍스트는 단정적 단언을 지양하고, 경향·가능성 중심 서술로 작성한다.

#### F6-1. 성격·기질 해석
- 일간 오행·십신 구성 기반 기질 분석.
- 강점·약점 서술.
- 인생 주요 키워드 3–5개 도출.

#### F6-2. 직업·재물 해석
- 용신 오행 기반 적성 직업군 안내.
- 재성(財星) 분석 기반 재물 운 방향 서술.
- 사업 vs 직장 성향 분석.

#### F6-3. 건강 해석
- 오행 과불급(過不及) 기반 취약 신체 부위 안내.
- 건강 관리 방향 제안.
- 의료 조언이 아님을 명시하는 면책 문구 포함.

#### F6-4. 인간관계 해석
- 관성(官星)·인성(印星) 기반 대인관계 패턴 서술.
- 궁합에 유리한 일간 오행 방향 안내 (단, 특정인 지정 불가).

### F7. 부가 기능

#### F7-1. 결과 저장 및 공유
- 분석 결과를 고유 URL로 생성하여 공유 가능.
- 공유 URL은 입력 데이터를 쿼리 파라미터 또는 해시로 인코딩 (서버 저장 없이).
- 링크 복사 버튼 제공.

#### F7-2. 결과 인쇄(PDF)
- 브라우저 인쇄 최적화 CSS 적용.
- 인쇄 시 命盤 + 오행 분포 + 대운 타임라인 + 종합 해석 포함 단일 페이지 레이아웃.

#### F7-3. 사주 용어 사전
- 앱 내 주요 용어(천간·지지·오행·십신·대운 등) 툴팁 또는 전용 사전 페이지 제공.
- 한자 용어 클릭 시 한글 설명 팝업.

## 5. 비기능 요구사항

### 5.1 성능
- 사주 산출 및 전체 결과 렌더링: 입력 완료 후 **1초 이내** 표시.
- 페이지 최초 로딩(LCP): **2.5초 이하** (3G 네트워크 기준).
- 폰트 로딩으로 인한 레이아웃 시프트(CLS): **0.1 이하**.

### 5.2 정확성
- 만세력 데이터: 1900–2100년 절기 정밀 데이터 내장.
- 절입 시각: 분 단위까지 반영.
- 음력 변환: 국립천문연구원 기준 데이터 활용.

### 5.3 호환성
- 지원 브라우저: Chrome 100+, Safari 16+, Firefox 110+, Edge 100+.
- 모바일 반응형: iOS Safari, Android Chrome 완전 지원.
- 최소 뷰포트: 320px 이상.

### 5.4 접근성
- WCAG 2.1 AA 준수.
- 스크린 리더: 한자 장식 요소 전체 `aria-hidden="true"` 처리.
- 키보드 탐색: 모든 인터랙티브 요소 포커스 탐색 가능.
- 색상 단독 정보 전달 금지 — 오행 색상에 반드시 한글 텍스트 병기.

### 5.5 개인정보
- 사용자 생년월일 등 개인정보를 서버에 저장하지 않는다.
- 모든 계산은 클라이언트(브라우저) 측에서 실행.
- 공유 URL은 클라이언트 사이드 인코딩만 사용.
